Como instalar:

TWRP:

Após instalar a rom, vá em:

  - 1° Install
  - 2° Selecione o fix (kernel)
  - 3° Deslize a barra ate o final
  - 4° Reboot system

CWM:

 - Após instalar a rom, vá em:

  - 1° Install Zip
  - 2° Choose zip from SDcard1
  - 3° Selecione o fix (kernel)
  - 4° Confirme (yes)
  - 5° Reboot system now
